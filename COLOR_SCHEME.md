## 🎯 مبادئ التصميم

### 1. التباين العالي
- النصوص الأساسية لها تباين عالي مع الخلفية
- سهولة القراءة على جميع الأجهزة

### 2. الاتساق
- استخدام نفس الألوان في جميع الشاشات
- تجربة موحدة للمستخدم

### 3. الحداثة
- ألوان عصرية وجذابة
- تتماشى مع أحدث الاتجاهات

### 4. الوصول
- توافقية عالية مع ضعاف البصر
- نسب تباين WCAG AA

## 🌙 الوضع الليلي (Dark Mode)

### الخصائص:
- ✅ ألوان أفتح للنصوص
- ✅ خلفيات داكنة لراحة العين
- ✅ تباين محسّن
- ✅ توفير البطارية على الأجهزة OLED

### الاستخدام:
```typescript
// في المكونات
const colors = useColors();
// سيتم تطبيق الألوان المناسبة تلقائياً
```

---

## 📱 أمثلة الاستخدام

### زر أساسي
```tsx
<TouchableOpacity 
  style={{ backgroundColor: colors.primary }}
>
  <Text style={{ color: colors.background }}>
    اضغط هنا
  </Text>
</TouchableOpacity>
```

### بطاقة منتج
```tsx
<View style={{ 
  backgroundColor: colors.surface,
  borderColor: colors.border,
  borderWidth: 1
}}>
  <Text style={{ color: colors.foreground }}>
    عنوان المنتج
  </Text>
  <Text style={{ color: colors.muted }}>
    الوصف
  </Text>
</View>
```

### رسالة نجاح
```tsx
<View style={{ backgroundColor: colors.success }}>
  <Text style={{ color: colors.background }}>
    تم بنجاح!
  </Text>
</View>
```

## 🎨 تخصيص الألوان

### لتغيير الألوان:
1. عدّل `theme.config.js`
2. أضف الألوان الجديدة
3. استخدمها في المكونات عبر `useColors()`

### مثال:
```javascript
const themeColors = {
  primary: { light: '#YOUR_COLOR', dark: '#YOUR_COLOR' },
  // ...
};
```
