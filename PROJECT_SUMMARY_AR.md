## كيفية التشغيل

### المتطلبات الأساسية
- Node.js (v18 أو أحدث)
- pnpm (أو npm/yarn)
- Expo Go على هاتفك المحمول (اختياري للتجربة على جهاز حقيقي)

### خطوات التثبيت
1. قم بتثبيت التبعيات:
   ```bash
   pnpm install
   ```
2. قم بإعداد ملف البيئة `.env` (استخدم `.env.example` كمرجع).
3. قم بتشغيل خادم التطوير (Server & Metro):
   ```bash
   pnpm dev
   ```

### تشغيل قاعدة البيانات
التطبيق يستخدم Drizzle ORM. لتحديث قاعدة البيانات:
```bash
pnpm db:push
```
