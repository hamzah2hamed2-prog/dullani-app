# خريطة التنقل الشاملة لتطبيق دلني

## نظرة عامة على التطبيق

تطبيق **دلني** هو منصة تسوق اجتماعية ومحلية تربط بين المستهلكين والمتاجر التجارية. يتكون التطبيق من قسمين رئيسيين:

1. **تطبيق المستهلك** - للبحث عن المنتجات والمتاجر
2. **تطبيق التاجر** - لإدارة المتاجر والمنتجات

---

## الأقسام الرئيسية للمستهلك

### 1. الاستكشاف (Home)
**المسار:** `/(tabs)/index`

**الميزات:**
- عرض شبكة المنتجات المميزة
- عرض الفئات الأفقية
- البحث السريع
- عرض العروض الخاصة
- التنقل إلى تفاصيل المنتج

**الأزرار الرئيسية:**
- 🔍 زر البحث → ينقل إلى شاشة البحث
- ❤️ زر المفضلة → يضيف/يزيل المنتج من المفضلة
- ➡️ زر التفاصيل → ينقل إلى تفاصيل المنتج

---

### 2. البحث (Search)
**المسار:** `/(tabs)/search`

**الميزات:**
- البحث النصي عن المنتجات
- تصفية حسب الفئات
- تصفية حسب السعر
- البحث بالصور
- عرض المتاجر على الخريطة
- سجل البحث

**الأزرار الرئيسية:**
- 📷 زر البحث بالصور → ينقل إلى شاشة البحث بالصور
- 🗺️ زر المتاجر → ينقل إلى خريطة المتاجر
- 🔍 زر البحث → يبحث عن المنتجات

---

### 3. المفضلة (Wishlist)
**المسار:** `/(tabs)/wishlist`

**الميزات:**
- عرض المنتجات المفضلة
- حذف المنتجات من المفضلة
- الانتقال إلى تفاصيل المنتج
- عدد المنتجات المفضلة

**الأزرار الرئيسية:**
- ❤️ زر المفضلة → يحذف المنتج من المفضلة
- ➡️ زر التفاصيل → ينقل إلى تفاصيل المنتج

---

### 4. الملف الشخصي (Profile)
**المسار:** `/(tabs)/profile`

**الميزات:**
- عرض بيانات المستخدم
- تعديل الملف الشخصي
- عرض الاهتمامات
- الوصول إلى لوحة تحكم التاجر (للتجار)
- تسجيل الخروج

**الأزرار الرئيسية:**
- ✏️ زر التعديل → ينقل إلى شاشة تعديل الملف الشخصي
- 📊 زر لوحة التحكم → ينقل إلى لوحة تحكم التاجر (للتجار فقط)

---

## الشاشات الفرعية للمستهلك

### تفاصيل المنتج
**المسار:** `/(tabs)/product/[id]`

**الميزات:**
- عرض صور المنتج (Carousel)
- عرض الاسم والسعر والوصف
- عرض معلومات المتجر
- عرض التقييمات والمراجعات
- زر المفضلة
- زر التواصل عبر واتساب
- زر الانتقال إلى ملف المتجر

**الأزرار الرئيسية:**
- ❤️ زر المفضلة → يضيف/يزيل من المفضلة
- 💬 زر واتساب → يفتح محادثة واتساب
- 🏪 زر المتجر → ينقل إلى ملف المتجر

---

### ملف المتجر
**المسار:** `/(tabs)/store/[id]`

**الميزات:**
- عرض معلومات المتجر
- عرض ساعات العمل
- عرض رقم الهاتف والعنوان
- عرض جميع منتجات المتجر
- عرض التقييمات
- زر الموقع على الخريطة

**الأزرار الرئيسية:**
- 📞 زر الاتصال → يفتح تطبيق الهاتف
- 🗺️ زر الخريطة → ينقل إلى خريطة الموقع
- ➡️ زر المنتج → ينقل إلى تفاصيل المنتج

---

### خريطة الموقع
**المسار:** `/(tabs)/store/[id]/map`

**الميزات:**
- عرض الموقع على الخريطة
- عرض العنوان الكامل
- زر الحصول على الاتجاهات
- معلومات الاتصال

**الأزرار الرئيسية:**
- 🧭 زر الاتجاهات → يفتح تطبيق الخرائط

---

### البحث بالصور
**المسار:** `/(tabs)/image-search`

**الميزات:**
- التقاط صورة من الكاميرا
- اختيار صورة من المعرض
- عرض نتائج البحث

**الأزرار الرئيسية:**
- 📷 زر الكاميرا → يفتح الكاميرا
- 🖼️ زر المعرض → يفتح معرض الصور

---

### خريطة المتاجر
**المسار:** `/(tabs)/stores-map`

**الميزات:**
- عرض جميع المتاجر في قائمة
- عرض معلومات المتجر
- الانتقال إلى ملف المتجر

**الأزرار الرئيسية:**
- ➡️ زر المتجر → ينقل إلى ملف المتجر

---

### تعديل الملف الشخصي
**المسار:** `/(tabs)/profile-edit`

**الميزات:**
- تعديل الاسم والبريد الإلكتروني
- إضافة السيرة الذاتية
- اختيار الاهتمامات والفئات المفضلة
- حفظ التغييرات

**الأزرار الرئيسية:**
- 💾 زر الحفظ → يحفظ التغييرات
- ❌ زر الإلغاء → يلغي التغييرات

---

### إعادة تعيين كلمة المرور
**المسار:** `/forgot-password`

**الميزات:**
- إدخال البريد الإلكتروني
- إرسال رابط إعادة التعيين
- إعادة تعيين كلمة المرور

**الأزرار الرئيسية:**
- 📧 زر الإرسال → يرسل رابط إعادة التعيين

---

## الأقسام الرئيسية للتاجر

### لوحة تحكم التاجر
**المسار:** `/(tabs)/merchant/dashboard`

**الميزات:**
- عرض الإحصائيات العامة
- عرض المنتجات مع المشاهدات والنقرات
- عرض معدل التحويل
- تبويبات للتنقل

**الأزرار الرئيسية:**
- ➕ زر إضافة منتج → ينقل إلى شاشة إضافة منتج
- ✏️ زر تعديل → ينقل إلى شاشة تعديل المنتج
- 🗑️ زر حذف → يحذف المنتج

---

### إضافة منتج
**المسار:** `/(tabs)/merchant/add-product`

**الميزات:**
- اختيار صورة المنتج
- إدخال بيانات المنتج
- اختيار الفئة
- إدخال السعر والوصف

**الأزرار الرئيسية:**
- 📷 زر الصورة → يفتح معرض الصور
- ✅ زر الحفظ → يحفظ المنتج

---

### تعديل المنتج
**المسار:** `/(tabs)/merchant/edit-product/[id]`

**الميزات:**
- تعديل بيانات المنتج
- تعديل الصورة
- حذف المنتج

**الأزرار الرئيسية:**
- 💾 زر الحفظ → يحفظ التغييرات
- 🗑️ زر الحذف → يحذف المنتج

---

### تعديل بيانات المتجر
**المسار:** `/(tabs)/merchant/edit-store`

**الميزات:**
- تعديل اسم المتجر
- تعديل العنوان والموقع
- تعديل رقم الهاتف
- تعديل ساعات العمل
- إضافة الوصف

**الأزرار الرئيسية:**
- 💾 زر الحفظ → يحفظ التغييرات

---

## تدفق المستخدم الرئيسي

### 1. تسجيل الدخول والتسجيل
```
تطبيق البداية
    ↓
شاشة تسجيل الدخول (OAuth)
    ↓
إعداد الملف الشخصي (أول مرة)
    ↓
اختيار الاهتمامات
    ↓
الشاشة الرئيسية (الاستكشاف)
```

### 2. البحث عن المنتجات
```
الاستكشاف
    ↓
البحث (نصي أو بالصور)
    ↓
نتائج البحث
    ↓
تفاصيل المنتج
    ↓
ملف المتجر أو واتساب
```

### 3. إدارة المتجر (للتجار)
```
الملف الشخصي
    ↓
لوحة تحكم التاجر
    ↓
إضافة/تعديل/حذف منتج
    ↓
عرض الإحصائيات
```

## ملاحظات مهمة

1. **التنقل بين الشاشات:** جميع الشاشات متصلة بشكل منطقي وسهل الاستخدام
2. **زر العودة:** متوفر في جميع الشاشات الفرعية
3. **شريط التنقل السفلي:** ثابت في جميع الشاشات الرئيسية
4. **الإشعارات:** متاحة من أي شاشة عبر زر الإشعارات في رأس الشاشة
5. **المفضلة:** يمكن إضافة/إزالة من أي شاشة تعرض المنتجات
