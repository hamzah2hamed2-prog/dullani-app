## 🔄 مخطط التنقل

### تدفق المستهلك (Consumer Flow)
```
تسجيل الدخول
    ↓
اختيار نوع الحساب (مستهلك)
    ↓
إعداد الملف الشخصي
    ↓
الشاشة الرئيسية
    ├─ البحث (محدثة)
    ├─ قائمة الرغبات
    ├─ المتابعات
    ├─ الملف الشخصي
    └─ تفاصيل المنتج
        ├─ ملف المتجر
        ├─ خريطة المتجر
        └─ التعليقات
```

### تدفق التاجر (Merchant Flow)
```
تسجيل الدخول
    ↓
اختيار نوع الحساب (تاجر)
    ↓
إعداد الملف الشخصي
    ↓
الملف الشخصي
    ├─ لوحة التحكم
    ├─ إضافة منتج
    ├─ تعديل منتج
    ├─ تعديل المتجر
    └─ الإشعارات
```
