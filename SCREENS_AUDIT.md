## 🔄 مخطط التنقل بين الشاشات

### تدفق المستخدم العام (Consumer)

```
تسجيل الدخول
    ↓
اختيار نوع الحساب (مستهلك)
    ↓
إعداد الملف الشخصي
    ↓
الشاشة الرئيسية
    ├─→ البحث
    ├─→ قائمة الرغبات
    ├─→ المتابعات
    ├─→ الملف الشخصي
    └─→ تفاصيل المنتج
        ├─→ ملف المتجر
        │   ├─→ خريطة المتجر
        │   └─→ منتجات أخرى
        ├─→ التعليقات
        └─→ مشاركة/إعجاب
```

### تدفق التاجر (Merchant)

```
تسجيل الدخول
    ↓
اختيار نوع الحساب (تاجر)
    ↓
إعداد الملف الشخصي
    ↓
الملف الشخصي
    ├─→ لوحة التحكم
    ├─→ إضافة منتج
    ├─→ تعديل منتج
    ├─→ تعديل المتجر
    └─→ الإشعارات
```
